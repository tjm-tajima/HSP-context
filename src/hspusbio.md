## 命令詳細

### uio_find

USB-IOを検索・初期化

- **グループ**: USBデバイス制御命令

**構文**
```
uio_find
```

**パラメータ**
- なし

**説明**

ディバイスの中からUSB-IOを検索・初期化します．
失敗するとシステム変数statにエラーコードが帰ります．
```
  stat : エラー内容
 ----------------------------------------------------
   0   : 正常
   1   : 必要なドライバが無い(まだUSB-IOを繋いだことが無い場合など)
   2   : USB-IOが繋がってない
```

**関連**: uio_free

---

### uio_free

USB-IOを解放

- **グループ**: USBデバイス制御命令

**構文**
```
uio_free
```

**パラメータ**
- なし

**説明**

USB-IOを解放します．
HSPの終了時に自動的に呼び出されますが，手動で呼び出しても害はありません．

**関連**: uio_find

---

### uio_getdevs

USB-IOの接続数を取得

- **グループ**: USBデバイス制御命令

**構文**
```
uio_getdevs
```

**パラメータ**
- なし

**説明**

繋がれているUSB-IOの数をシステム変数statに返します．

**関連**: uio_seldev

---

### uio_seldev

複数のUSB-IOから選択

- **グループ**: USBデバイス制御命令

**構文**
```
uio_seldev p1
```

**パラメータ**
- `p1(0)`: 番号(0～uio_getdevsで得た値)

**説明**

複数のUSB-IOが繋がれている場合に何番目を使うかを指定します．

**関連**: uio_getdevs

---

### uio_out

USB-IOデータ出力

- **グループ**: USBデバイス制御命令

**構文**
```
uio_out p1,p2,p3
```

**パラメータ**
- `p1(0)`: 出力するI/Oポートアドレス(0～1)
- `p2(0)`: 出力する内容(8ビット)
- `p3(0)`: モード

**説明**

USB-IOに任意のデータを出力します．
失敗するとシステム変数statに1が返ります．
p3に1を指定すると，ポート1のp1にパルスを送りポート0を書き換えます．

**関連**: uio_inp

---

### uio_inp

USB-IOデータ読み込み

- **グループ**: USBデバイス制御命令

**構文**
```
uio_inp p1,p2,p3
```

**パラメータ**
- `p1`: データを読み込む変数
- `p2(0)`: I/Oポートアドレス(0～1)
- `p3(0)`: モード

**説明**

USB-IOからデータを入力します．
失敗するとシステム変数statに0以外の値が返ります．
p3に1を指定すると，ポート1のp1にパルスを送りポート0から読み込みます．

**関連**: uio_out
