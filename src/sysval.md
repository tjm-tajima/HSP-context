## 命令詳細

### hspstat

HSPランタイムの情報

- **グループ**: システム変数

**構文**
```
hspstat
```

**パラメータ**
- なし

**説明**

HSPランタイムの情報が整数値で代入されます。以下の情報がすべて合計された値になります。
 デバッグモード = 1
 スクリーンセーバー起動時 = 2
 コンソールモード = 16
 Macintosh版HSP = $80
 HSP3Dish = $100
 Linux版HSP = $1000
 UTF8文字列ランタイム = $20000
 64ビットランタイム = $40000

**関連**: _debug, ;---------------------------------------------------------------------

### iparam

割り込み要因を示す値

- **グループ**: システム変数

**構文**
```
iparam
```

**パラメータ**
- なし

**説明**

割り込み要因を示す値が代入されます。
oncmd命令による割り込み時は、割り込みメッセージのIDが代入されます。
onexit命令による終了割り込み時は、通常のウィンドウクローズによる終了であれば0が、シャットダウン等のシステム通知による終了であれば1が代入されます。

**関連**: onkey, onclick, onexit, lparam, wparam, onerror, oncmd

**サンプル**
```hsp
	onkey goto *lab
	stop
*lab
	mes "キーコード : "+wparam+", 文字コード : "+iparam
	stop
%port-
Let

;---------------------------------------------------------------------
```
